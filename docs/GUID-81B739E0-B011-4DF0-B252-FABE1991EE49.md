# ZGPL_ResetTableEntry Function

## C

```c
void ZGPL_ResetTableEntry(void *entry, ZGP_EntryType_t tableType);
```

## Description

 To reset all the parameters of the given entry to init value

## Parameters

| Param | Description |
|:----- |:----------- |
| entry | table entry pointer
|tableType | proxy or sink 

## Returns

 None
