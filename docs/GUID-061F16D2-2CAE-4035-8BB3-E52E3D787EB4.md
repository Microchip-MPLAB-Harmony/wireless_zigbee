# ZGPL_SetDeviceMode Function

## C

```c
void ZGPL_SetDeviceMode(bool isProxy, ZGP_Mode_t mode);
```

## Description

 To set the Device in operational/Commissioning Mode

## Parameters

| Param | Description |
|:----- |:----------- |
| isProxy | is proxy or sink device 
| mode | operational/Commissioning 

## Returns

  None
