5.2.1.10 APS_RemoveAllGroupsReq_t Structure

Syntax
typedef struct
{
 /** The endpoint from which the group is being removed. */
 Endpoint_t endpoint;
 /** The confirm primitive as transaction result. */
 APS_RemoveAllGroupsConf_t confirm;
} APS_RemoveAllGroupsReq_t;

Description

The APS_RemoveAllGroupsReq_t structure, defined in the apsmeGroup.h and aps.h files, specifies the parameters for the APS_RemoveAllGroupsReq() function. This function requires these parameters to remove all groups.
