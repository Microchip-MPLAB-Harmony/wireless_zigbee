2.3 BitCloud Stack APIs

The stack API functions fall into the categories related to the following layers:

  • Base Device Behavior (BDB)
  • Configuration Server (CS) functions
  • Hardware Abstraction Layer (HAL) (functions dealing with the hardware functionality of the MCU)
  • System environment
  • Zigbee Cluster Library (ZCL)
  • Zigbee Device Object (ZDO)
  • Zigbee Green Power (ZGP)
  • Network (NWK)
