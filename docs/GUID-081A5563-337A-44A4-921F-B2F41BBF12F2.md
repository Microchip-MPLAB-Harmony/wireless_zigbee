# ZGPL_GetDeviceMode Function

## C

```c
ZGP_Mode_t ZGPL_GetDeviceMode(bool isProxy);
```

## Description

 To get the Device Mode (operational/Commissioning Mode)

## Parameters

| Param | Description |
|:----- |:----------- |
| isProxy | true if the device is proxy, false for sink  

## Returns

 CurrentMode of Device
